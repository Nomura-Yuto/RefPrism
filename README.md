# 【概要】
- **制作環境：** DirectX11 / C++  
- **制作人数：** 1人  
- **制作期間：** 6ヵ月 (2024/09 ～ 2025/02)  


# 【ゲーム詳細】
フィールド中央からプレイヤーに向かって常にレーザーが放たれいる。  
プレイヤーは反射と屈折の状態を切り替えることができ、その状態によりレーザーの軌道も変わる。  
反射・屈折の状態をうまく切り替えながら敵を倒しスコアを稼いでいくゲーム。
